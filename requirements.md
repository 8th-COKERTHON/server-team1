# 프로젝트 기능 명세 (온보딩 파트)

- [ ] 디바이스 아이디(deviceId)는 필수값이다.
- [ ] 닉네임(nickname)은 2자 이상 8자 이하여야 한다.
- [ ] 유저 생성 시 상태값은 "LOGIN" 또는 "SIGN_UP"으로 구분해야 한다.
- [ ] 에러 발생 시 `ErrorCode` Enum을 사용하여 처리해야 한다.
- [ ] API 응답은 `ApiResponse` 형식을 따라야 한다.
